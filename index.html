Kako postaviti ovaj portal online besplatno 24/7
Ovaj portal je jedna HTML datoteka – ne treba server, baza podataka ni hosting plan. Postoji više besplatnih opcija.
Preporučene besplatne opcije
GitHub Pages PREPORUČENO
Besplatno zauvek
Microsoft platforma, 100% besplatna za statičke sajtove. Bez limita propusnosti. SSL sertifikat gratis. URL: korisnik.github.io/windows-os. Potreban GitHub nalog.
Netlify
Besplatno (100GB/mes)
Prevuci i pusti HTML fajl. Gotovo za 30 sekundi. Automatski HTTPS. URL: ime.netlify.app. Moguć i custom domen.
Cloudflare Pages
Besplatno (bez limita)
Najbrži CDN na svetu. Neograničen bandwidth. URL: ime.pages.dev. Zahteva GitHub nalog za deploy.
Korak-po-korak: GitHub Pages (najlakši put)

Kreirajte GitHub nalog

Idite na github.com → Sign up → unesite email, lozinku i korisničko ime (npr. profos-pera). Besplatno, bez kreditne kartice.

Kreirajte novi repozitorijum

Kliknite New repository → unesite ime npr. windows-os-portal → označi Public → Create repository.

Otpremite HTML fajl

Na stranici repozitorijuma kliknite uploading an existing file → prevucite os_portal.html → obavezno preimenujte u index.html → Commit changes.

Fajl mora biti nazvan: index.html

Uključite GitHub Pages

U repozitorijumu: Settings → Pages → Source → Deploy from branch → main → / (root) → Save. Sačekajte 1–2 minuta.

Vaš portal je online!

URL će biti: https://korisnicko-ime.github.io/windows-os-portal/
Pošaljite link učenicima – radi na svakom uređaju, telefonu, tabletu, računaru.

Primer: https://profos-pera.github.io/windows-os-portal/

Ažuriranje sadržaja

Kad dobijete novu verziju HTML fajla, jednostavno otpremite novi fajl na GitHub (zamena starog). Sajt se automatski ažurira za ~1 minut. Bez FTP, bez servera.

(Opciono) Vlastiti domen

Ako škola ima domen (npr. ostrakomp.rs), možete podesiti da portal bude na adresi os.ostrakomp.rs — besplatno putem CNAME DNS zapisa. GitHub Pages podržava custom domene bez doplate.

Settings → Pages → Custom domain → unesite domen → Save
Napomena o nastavničkoj lozinci
Lozinka je ugrađena u HTML fajl. Za produkcijsku upotrebu promenite je pre postavljanja online:
U fajlu pronađite: TEACHER_PASSWORD = 'nastavnik2025'
i zamenite sa sopstvenom lozinkom pre upload-a na GitHub.
